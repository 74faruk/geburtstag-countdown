# Geburtstags-Countdown

Zeigt das aktuelle Alter und einen Live-Countdown bis zum nächsten
Geburtstag.

## Features

- Live-Countdown (Tage, Stunden, Minuten, Sekunden)
- Aktuelles Alter
- Wochentag der Geburt

## Tech

Reines HTML, CSS und JavaScript — kein Framework, kein Build-Schritt.

## Lokal öffnen

`index.html` einfach im Browser öffnen, fertig.
